从9月6日开始，进行为期10天的Python学习，作为自己博客网站的第一个小项目。对于自己学习Python,做一些整理。

第一天的学习,主要是包括变量的，运算符，和数据类型，位运算。任何一门语言，都要了解一些最基础的语法知识。因此，以变量为基本单元，进行逐步深入学习。

👉 变量

在高级程序语言设计中，变量都是最基础的组成单位。对于变量，只需要掌握变量命名的三个规则：

  1. 变量可以用字母，符号，数字组成；
  2. 数字不能做为变量的开头，符号，字母可以做为变量的开头
  3. 变量不能与系统保留字冲突

👉 数据类型

整型例子：

a = 1024
b = dir(int)
print(a,type(a) ) 可以生成 变量的类型
print(bin(a))      获取 数字的二进制表示
print(b)

浮点数类型

import decimal
from decimal  import Decimal

a = decimal.getcontext()
print(a)

b = Decimal(1)/Decimal(3)
print(b)

decimal.getcontext().prec = 4

print(c)
